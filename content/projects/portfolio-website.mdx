## Overview

Designed and developed my personal portfolio website to showcase projects, publish blog posts, and host interactive AI demos — all from a single, statically-generated Next.js application.

## Features

- **MDX-powered content** — Write blog posts and project descriptions in Markdown with embedded React components
- **Dark mode** — System-aware theme toggle with smooth transitions
- **AI Playground** — Embedded Hugging Face Spaces and Streamlit apps for interactive demos
- **Responsive design** — Mobile-first layout that looks great on all devices
- **SEO optimized** — Dynamic metadata, sitemap, and Open Graph tags for every page

## Technical Decisions

- Chose **Next.js App Router** for its built-in static generation and file-based routing
- Used **Tailwind CSS v4** with CSS-based theming for design tokens
- Implemented **Framer Motion** for subtle scroll-triggered animations
- Content stored as **MDX files** in the repo — no CMS or database needed

## Lessons Learned

Building a portfolio is a great exercise in restraint. The biggest challenge was resisting the urge to over-engineer and keeping the focus on content and presentation.
